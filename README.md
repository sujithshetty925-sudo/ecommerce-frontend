#Ecommerce-frontend
##Overview
This project is the frontend of an E-commerce web applications built using React.It provides a modern and user-friendly interface for browsing ,products,managing the shopping cart,placing orders,and viewing user information.

## Tech Stack
-React.js
-Javscipt
-HTML5
-CSS3
-Axios
-React Router Dom

## Features
-Home Page
-Product Listing
-Product Search
-Product Details
-Add to Cart
-Buy Now
-Checkout Page
-Payment Page(demo)
-Order Success page
-My Orders
-User Profile
-Responsive Navigation Bar
-Modern Footer
-Responsive UI

##Project Structure
#src
 -components
 -pages
 -services
 -assets
 -App.js
 -Index.js

## How to Run

1.Open project in the vscode
2.Install dependencies:
`Bash npm install`
3.Start the React application:
 `bash npm start`
4.open your browser and visit:
 http://loaclhost:3000

##Backend
This frontend communicate with Spring Boot REST API backend for product and order management.
